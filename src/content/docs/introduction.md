# Welcome to Clarity

Clarity is a modern, feature-rich documentation platform built with Astro 5. It's designed to help teams create beautiful, searchable, and maintainable documentation with minimal setup.

## Why Clarity?

**Built for Teams**: Clarity provides authentication, role management, and collaboration features out of the box.

**Developer-Friendly**: Write documentation in Markdown, with full TypeScript support and hot reload during development.

**Production-Ready**: Includes OAuth authentication (Google & GitHub), full-text search, SEO optimization, and responsive design.

**Customizable**: Configure everything through a single `clarity.config.ts` file - from branding to feature toggles.
